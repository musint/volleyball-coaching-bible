# Munciana Team Out of System (Wing Rotation)

## Setup
Wings: a left front and right front on each side. Two back-row defenders behind each wing. Everyone else fills behind the wings off-court. Coach with a cart sideline. **No middles** — this is a wing-only out-of-system drill.

## Execution
1. Coach hits or tosses a hard ball to a back-row defender.
2. The defender who digs is **not** the setter — the **other** back-row defender sets.
3. Both back-row defenders then **move forward to attack**. The original two front-row hitters become the new diggers in the back row.
4. Both hitters always shag and exit; the team rotates through positions on every rep: dig → set → become the next attacker; old hitters become the next diggers.
5. **[00:38:32]** "DS whoever defends moves up to attack." Cycle is dig → set → both defenders shift up to hit; both old hitters drop to defense.
6. Coach enters the next ball immediately so the rotation never stops.

## Coaching points
- This is the **most live** out-of-system drill in the library. **[00:39:14]** "I don't do this drill often. But it is probably the best one for actually playing live balls."
- Cover hard on every swing; the rotation depends on bodies moving forward into attack as soon as their teammate is the one setting.
- Communication is constant: every defender announces who's setting and who's hitting on the next ball before the contact happens.
- Pace: coach enters the next ball within a beat of the previous swing. The drill has to feel like a real rally chain, not a series of resets.

## Variations
- Add a single blocker against each attack so the swing is contested.
- Score it as a wash drill: team must win two rotations in a row to bank a point.
- Limit attack to back-row only — same rotation, all attacks come out of the back row.

## Adaptations by level
- **14U**: not appropriate as written — too many simultaneous role transitions. Use [[munciana-mia-drill]] or [[munciana-two-man-out-of-system]] instead.
- **HS**: standard rules; coach enters every 4-6 seconds.
- **College**: add blockers, full coverage, and run continuously for 90 seconds before any reset.
