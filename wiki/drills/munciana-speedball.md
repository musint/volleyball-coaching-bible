# Speedball

## Setup
Two even teams across the net, full court. Catch-and-throw only — no rebounds, no contacts. Mike's setup call: "All catch and throw. All catch and throw" **[00:12:14]**. Mike opens the defense block by naming this drill the entry: "Speedball is the intro to defense. We want people to learn to read and run" **[00:12:29]**.

## Execution
1. Player catches the ball anywhere on the court and throws it back over the net.
2. Throw must land in for a point: "1-0, we score it every time. 2-0... 2-1, go" **[00:12:43]**.
3. Games are short — Mike: "we tend to play minute and a half games" **[00:12:53]**.
4. Continuous flow: as soon as the ball lands or is caught, it goes back; Mike pushes the cadence with a constant "go."
5. The lesson: "I want them to learn to read and run. And this is a simple read and run game" **[00:13:11]**.

## Coaching points
- The whole point is read-and-move — Mike's diagnostic: "most of the time your guys are watching and standing. So we want to read and run" **[00:12:34]**.
- No standing while watching — anyone caught flat-footed is the next defender beat.
- Catch position matters: catch the ball where you would have dug it, not where it lands easiest.
- Use as the day's first defense rep so reading-and-running becomes the warmed-up default for every later drill.

## Variations
- Add bounces (two-bounce rule) for younger groups so the ball stays in play.
- Score by floor-touches per side.
- Run as 3v3 on a single side for tight-court reading reps.

## Adaptations by level
- **14U:** catch-and-throw only, generous boundaries; 1.5-minute games.
- **HS:** add a one-touch-then-throw rule to bridge into real defense.
- **College:** allow a single overhead set as the second contact, then throw — closer to live transition.
