# Serve Receive Competition

## Setup
One server on side A, three passers plus a target at the setter position on side B. A second trio waits to sub in on the sideline. Scoreboard with two rows: SERVER and PASSERS. Ball cart behind the serving endline; shagger duty rotates with the losing side.

## Execution
1. Server delivers one serve per rep. Target calls pass grade aloud 0-3 (3 = any play; 2 = high outside only; 1 = out-of-system; 0 = ace/shank).
2. **Server scores** on any 0 or 1 pass. **Passers score** on any 2 or 3 pass.
3. After the grade, coach or server comments on one of four dimensions — technique, placement, hand-to-ball contact, outcome. Keep the loop under 10 seconds.
4. Play to 7. Losing side swaps in the waiting trio or a fresh server. Winners stay.
5. Rotate through every server and every triad; track which matchups produced the highest totals per side.

## Coaching points
- Both sides are scored — the server isn't warming up passers and passers aren't catching reps. Every contact is competitive.
- Grade honestly. A lenient target builds fake-confident passers; calling "that was a 1" aloud lets the passer adjust.
- Server-side: don't only aim for aces. A 1-pass that forces a pin-set is a tactical win too.
- Passer-side: shuffle-not-reach, platform early, call the ball before it crosses.

## Variations
- **1v1+1**: one server, one primary passer, one target. Tightens individual accountability.
- **Rotation-live**: run the receive rotation you actually play in matches; tracks which rotations servers break.
- **Tough-serve only**: server must jump-float or aim seams — raises the passer bar.

## Adaptations by level
- **14U**: 2-passer receive, grade scale 0-2 only, play to 5; coach serves the first rotation to calibrate expectation.
- **HS**: standard rules above; rotate in both float and jump-float servers across the block.
- **College**: 3-passer libero-split receive, full 0-3 scale, play to 9; introduce rotation-live variant at midpoint.
