# Serve receive error percentage drill

## Summary
Tod Mattox's SR error-percentage drill scores each of 10 attempts as plus, minus, or wash. Benchmark goal: error rate under 50%.

## Key claims / ideas
- Score each attempt as plus, minus, or wash
- 10-attempt sample per measurement
- Target benchmark: error rate under 50%
- Gives passers motivation via visible benchmark
- Simple stat workflow for SR training
