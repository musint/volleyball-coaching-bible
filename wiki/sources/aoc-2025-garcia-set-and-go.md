# Set and go drill for high energy set training

## Summary
Austin Juniors coach Mike Garcia's "Set and Go" drill fuses setting, mental processing, and conditioning: 4 setters along the net + 2 rotating through crunches/perimeter-run. Pin-to-pin long sets in a continuous cycle. Liskevych quote: most setting is NOT static — foot movement + thinking ahead is critical. Builds long-set arm/wrist strength.

## Key claims / ideas
- Setting is rarely static — foot movement + next-play thinking is critical (Liskevych).
- Setting + fatigue + decision = match realism; conditioning drills should include all three.
- "Set and Go" structure: 4 setters on net alternating pin-to-pin long sets, 2 rotating through conditioning (crunches/twists + perimeter run), constant position substitution.
- Pin-to-pin reps build arm/wrist strength for long sets, opening outside-hitter angles.
- Advanced setters should develop setting opposite to the direction the MB is moving.

- Conditioning + cognitive load + setting
- Long set development
- Mental "think ahead" framing of setting
