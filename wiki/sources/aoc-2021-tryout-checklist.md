## Summary

15-point checklist for running a thoughtful, well-organized volleyball tryout: planning, philosophy, staff roles, drill menu, skill-level sorting, equipment, jump testing, parent meetings, and roster-notification logistics.

## Key claims / ideas

- Every tryout needs a written plan specifying skills, hustle, team situations, physical testing.
- Coaches must answer *what they're looking for* (players vs. athletes, current skill vs. potential) before designing drills.
- Evaluator coaches should rotate systematically, not wander.
- Include Coleman-McKenzie test (pass/set alternating >6ft for 25-50 reps) and Sits Test in skill battery.
- Separate skill levels quickly so best players see each other.
- Vertical jump testing adds a cheap proxy for athletic potential.
- Parent meetings should run concurrent with tryouts to communicate format, selection criteria, fees.
- Have a plan for notifying cut athletes — parents, coach, director present together.
