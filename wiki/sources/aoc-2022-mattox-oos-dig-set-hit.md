# Out-of-system drill: Digging, setting, hitting

## Summary
Mattox's 5-station OOS drill: tossers at net, two lines of diggers/setters side-by-side, attackers on left and right. Tosser "bowls" to digger, next-line player sets, pin attacks. Each digger-setter gets two touches (dig then set, then rotate). Add a middle blocker for eye-work reps — read set direction, shuffle toward block spot.

## Key claims / ideas
- Transitioning off a dig is trainable — use 5 stations (tossers, 2 dig/set lines, 2 pin lines).
- Each dig-setter gets a dig then a set then rotates — two touches per rep.
- Setter must cover the hitter after setting.
- Add a MB to read set direction and shuffle — teaches mid-block eye work under OOS conditions.

## Topics covered
- OOS dig-to-set-to-hit transition
- Setter cover habit
- Middle-blocker read-and-shuffle eye work
