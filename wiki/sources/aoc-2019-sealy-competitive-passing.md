## Summary

Competitive 3-passer + setter + catcher drill per side with alternating servers. First to 25 wins. Scoring: perfect pass = 2, perfect set = 2, server miss = 2 to passing side, "relatively accurate" = 1.

## Key claims / ideas

- Differential scoring (2 for perfect, 1 for accurate) makes accuracy pay more than merely "in play" passing.
- Punishing server misses rewards the passing side for drawing bad serves.
