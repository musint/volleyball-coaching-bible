# Setting drills: Game-like touches to improve location

## Summary
John Dunning (Stanford) outlines three Stanford setter-training formats: fixed-target drills (tosses from different back-court points, set into hoops at pin or dump), setting-off-a-dig (ripped-ball dig + teammate set to pin targets), and controlled-vs-game-like. Dunning's nuanced position: controlled drills have a place for younger players building feel; game-like drills come next.

## Key claims / ideas
- Fixed-target drills: move to ball + set into a target (hoop at pin) trains location precision.
- Setting-off-a-dig: two players near net, coach rips ball, one digs, the other sets to a pin target — trains the active ball.
- Controlled toss drills are NOT useless — they give younger players feel before graduating to live/game-like drills.
- Progression is: controlled → live off passes → live off digs.
