# Hybrid systems: 6-2 where the setter and middle sub

## Summary
Tod Mattox details a hybrid 6-2 where setters sub for middles (rather than the usual setter-for-opposite), pairing each setter permanently with one middle so that setter-middle timing benefits from repetition.

## Key claims / ideas
- Standard 6-2: setters sub for opposites.
- Alternative: sub setters for middles, creating two fixed setter-middle pairs.
- Each pair only works with each other, allowing rhythm and timing to deepen.
- Reduces timing variance because setter and middle don't have to adjust to multiple partners.
- Can improve match outcomes via higher-quality quick attack chemistry.
