# Why does keeping passing stats matter?

## Summary
Brennan Dean (Wave Volleyball) explains the standard 3-point pass rating: 3 = perfect, 2 = mediocre, 1 = emergency play required on second/third contact, 0 = unplayable. Passing averages inform lineup and starter decisions.

## Key claims / ideas
- Passing stats enable data-informed lineup and starter decisions
- Standard 3-point scale: 3 perfect / 2 mediocre / 1 emergency / 0 unplayable (ace)
- "1" means the 2nd and 3rd contacts will be emergency plays — poor but playable
- Over time, each player's average passing number measures efficiency
- Passing average drives playing time decisions

- 3-point pass rating scale
- Passing stat workflow
- Data-driven lineup decisions
- Efficiency metrics for passers
