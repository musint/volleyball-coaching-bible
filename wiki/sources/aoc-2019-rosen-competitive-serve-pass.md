# Making your serve/pass drills competitive

## Summary
Mark Rosen (Michigan) mirrors a real-game rule in practice: servers who force a bad pass get to serve again. Three passers + target vs. several servers. Tally server team's total points; switch roles; most points wins.

## Key claims / ideas
- Mirrors the game: servers who put receivers out of system often get to serve again
- Setup: 3 passers + 1 target vs. several servers across net
- If pass is perfect/near-perfect, that server is done
- If pass is inaccurate or unplayable, that server gets another serve (earns point)
- After all servers cycle, tally server team's combined points
- Flip roles, run again — high point total wins the drill
- Requires 8+ players, full court

## Topics covered
- Competitive serve/pass drill design
- Rewarding out-of-system serves
- Team scoring based on server pressure
- Game-like serve-pass interaction
