## Summary

This is AOC curricular content; defer to qualified AT/CSCS/MD on programming specifics. Marie Zidek (DePaul) points out the asymmetry: coaches teach jumping but rarely teach landing. A simple progression: stand on chair/staircase → drop down → cushion with bent knees → soft landing. Rehearse variations: two-foot land, one-foot land, lateral-jump land, one-foot jump to one-foot land.

## Key claims / ideas

- Coaches teach jumping but almost never teach landing — this is a correctable blind spot.
- Progression: start with a drop from a chair/staircase (not a jump) to isolate the landing mechanics.
- Teach cushioning with bent knees + soft touch-down.
- Rehearse all four combinations: 2-foot land, 1-foot land, lateral land, 1-foot jump → 1-foot land.
- Rationale is long-horizon joint protection, not short-term performance.
