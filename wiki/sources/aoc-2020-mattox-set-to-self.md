# Set to self drill

## Summary
Mattox's beginner progression: toss → catch, toss → set → catch, toss → 5 consecutive self-sets. Develops controlled setting touch before partner work.

## Key claims / ideas
- Simple 3-stage progression for first-time setters.
- Stage 1: underhand toss + catch (tracking).
- Stage 2: toss + one set + catch (touch).
- Stage 3: toss + five consecutive self-sets (control).

- Beginner setting progression
- Self-set touch development
