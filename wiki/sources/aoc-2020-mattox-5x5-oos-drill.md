# 5x5 out-of-system setting drill

## Summary
Mattox's 5x5 target defines the OOS set zone: 5 feet off the sideline, 5 feet off the net, HIGH so hitters can take big approaches. Coach bounces a ball; receiving team bump-sets into the 5x5 box; attacker swings aggressively. Successful reps earn rotations.

## Key claims / ideas
- Standard OOS set target: 5 ft from sideline, 5 ft off net, high.
- High ball buys approach time for the hitter.
- Criteria for a "successful" rep: set near the 5x5 area AND aggressive attack.
