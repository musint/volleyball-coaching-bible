# Improving setter-hitter timing through tempo and communication

## Summary
Blake Rawlins (Omaha Supernovas; Volleyball Club Nebraska) lays out a tempo-and-communication framework for the setter-hitter connection: time sets from release to contact to fix offensive tempo, have hitters call "slow/go/fast" based on pass quality so they regulate their own approach timing, and push setters to miss inside rather than tight or off-the-antenna because it preserves hittable range.

## Key claims / ideas
- "The earlier you can find the ball with your eyes, the more time you have to attack the ball."
- Timing the set from setter release to hitter contact quantifies offensive tempo and drives setter consistency.
- Hitters should call "slow / go / fast" on each pass so their approach speed matches the reality of the pass, not a default tempo.
- Target contact for pin hitters: three feet off the net, three feet off the sideline; miss inside rather than tight or outside.
- A right-handed left-side hitter can hit with range from inside the antenna but gets "trapped" if the set is too close to the pin.

## Topics covered
- Setter-hitter timing and tempo measurement
- Hitter-approach regulation via verbal calls
- Set location targets (distance from net, distance from sideline)
- Setter miss-tendency guidance (miss inside)
- Block-to-offense transition (second half of video)
