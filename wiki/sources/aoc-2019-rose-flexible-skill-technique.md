## Summary

Russ Rose (Penn State '79-'21) on balancing team-wide technique standards with respect for each player's previously learned skills. Core message: work ethic is the non-negotiable common standard.

## Key claims / ideas

- Some technique elements should be standardized team-wide; others should tolerate individual variance players bring in.
- The one standard that cannot vary across players is work ethic.
- A team with hard-work as a constant can excel regardless of talent gap.

## Topics covered

Team-wide standards; individual technique variance; work ethic as common denominator.
