# Shoestring drill for wider blocking hands

## Summary
A shoestring tied between the blocker's wrists acts as a constraint: keep the string taut and the hands stay wide, covering more net.

## Key claims / ideas
- Tie shoestring wrist-to-wrist.
- Objective: string stays as taut as possible (minimal slack).
- Constraint forces wide hands; narrowing hands slackens the string.
- Simple, equipment-light feedback loop for hand spread.

## Topics covered
- Wide hand placement in blocking.
- Constraint-based coaching.
- Solo/pair blocking drill.
