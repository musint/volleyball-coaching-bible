# Karch Kiraly on building a program culture

## Summary
Video-backed, minimal text body. A session with Karch Kiraly and Luka Slabe on program culture in the U.S. Women's National Team: identifying bedrock values, translating them into daily practice, and letting them shape planning, training organization, presentation, and feedback.

## Key claims / ideas
- "The number one job of everyone in the program is to be a learner" — culture is grounded in a shared learning posture.
- A program should be able to name its 1–2 bedrock values, not a long list.
- Culture is evidenced by daily behaviors on court and off, not by slogans.
- Planning, training organization, presentation, and feedback are concrete vehicles for culture.

## Topics covered
- Program culture and core values
- Coaching philosophy / leadership
- Team-building and learner identity
- Translating values into practice organization and feedback mechanics
