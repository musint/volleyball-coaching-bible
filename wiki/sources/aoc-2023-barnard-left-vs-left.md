# Head to head drill: left vs. left

## Summary
Australian Volleyroos coach Mark Barnard's 6v6 hitting drill forces both teams' left-side hitters to swing into a known block and dig, generalizing to any position matchup and doubling as a ranking tool.

## Key claims / ideas
- Free-ball entry; first set must go to the left-side hitter on both sides.
- If the rally survives both first swings, subsequent sets can go anywhere.
- Generalizes to middle-vs-middle and right-vs-right for any positional matchup.
- Used as a ranking drill to sort depth charts (e.g., outsides 1 through 7).
- Core pedagogical benefit: hitter has to score even though the block knows the set is going to them.

## Topics covered
- 6v6 hitter-matchup drill design
- Known-block shot selection
- Depth-chart ranking via competitive matchup
- Cross-position generalization of the drill
