# The passing ladder

## Summary
Christy Johnson-Lynch's passing ladder is a competitive passer-vs-server drill with promotion/relegation. Passer and server score against each other in 8-ball rounds, and the winner moves up a court while the loser drops down.

## Key claims / ideas
- Head-to-head passer vs. server creates real stakes for both skills.
- Ladder promotion/relegation across multiple courts adds cross-team competition.
- 8-ball rounds provide enough sample to reward consistency, not single reps.
- Minimum 4 players; scales up with more courts running parallel ladders.
