# Triples pass drill

## Summary
Tod Mattox's Triples Pass drills platform tilt and ball angle: three-player triangle — one tosses, one passes, one sets. Rotate roles every five balls. Core cue: face the ball, tilt the platform to target.

## Key claims / ideas
- Triangle of three players with toss / pass / set roles
- Rotate positions every 5 balls
- Core focus: face the ball, tilt the platform to target
- Trains redirecting the ball without turning the body
- Good beginner fundamentals drill

## Topics covered
- Platform tilt fundamentals
- Face-ball-tilt-platform cue
- Three-player rotation drill
- Beginner passing → setting chain
