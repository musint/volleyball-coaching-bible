## Summary

Laurie Eisler (University of Alberta) frames error correction around a diagnostic question — is the error physical, psychological, technical, or tactical? — before any fix is applied.

## Key claims / ideas

- No fix is possible until the error's root cause is correctly classified.
- Four root categories: physical, psychological, technical, tactical.
- Misclassifying root cause produces wasted correction effort.

## Topics covered

Error root-cause taxonomy; coaching diagnostic frame; correction methodology.
