# Hitting a slower push set on the outside

## Summary
Bishop's School coach Tod Mattox gives a footwork drill that pairs an outside hitter's approach cadence to the precise moment the ball sits in the setter's hands for a push set.

## Key claims / ideas
- Push-set cadence for the outside: soft right → hard left → step → close → jump → catch.
- Right-foot step cue fires just before the ball reaches the setter's hands off the pass/toss.
- Hard left cues when the ball is in the setter's hands.
- Setter tosses from middle-front and delivers antenna-high sets to simulate game set trajectory.
- Progression is catch-only first, then live hitting once timing is reliable.

## Topics covered
- Push-set outside approach timing
- Step-by-step approach cadence cues
- Catch-before-hit progression as timing isolation
- Drill structure for setter-hitter sync
