# Incorporating the back-row attack into your offense

## Summary
Beth Launiere (Utah '90-'25) walks coaches through the decision framework before adding a back-row attack to their system: practice-time cost, level-appropriate effectiveness, efficiency threshold, in-system vs outlet use, number of back-row set types, and the height/speed dial.

## Key claims / ideas
- Back-row attack only pays off with sufficient practice-time investment; don't add it casually.
- Questions to answer first:
  - How much practice time to dedicate?
  - Is it effective at my level?
  - Efficiency high enough to justify, or outlet only?
  - Use it in-system, OOS, or both?
  - How many types of back-row sets?
  - Height and speed of each?
- Treats back-row as a real offensive option only if the team commits the reps.

## Topics covered
- Back-row attack decision framework
- Practice-time budgeting for new plays
- In-system vs out-of-system back-row use
- Back-row set typology and parameters
