# Out-of-system offense and defense drills

## Summary
DePaul's Marie Zidek runs two pre-match OOS drills: (1) blockers start pinched then spread on coach's attack so libero sets OOS while blockers seal the line; (2) off-net sets attacker targets opponent setter, defense blocks/digs and returns ball for in-system swing.

## Key claims / ideas
- Drill 1: blockers pinch-then-spread; libero gets OOS set reps; defenders learn OOS positioning.
- Drill 2: coach feeds off-net set to LS hitter; attacker targets opposing setter in back row.
- Drill 2 bounces between OOS (first ball) and in-system (second ball) attack.
- Good pre-match tune-up for OOS flow between offense and defense.

## Topics covered
- OOS offense and defense integration.
- Libero setting from OOS.
- High cut shots to Zones 1/6.
- Pinch-and-spread block pattern.
