# Perfect pass competition drill

## Summary
Tod Mattox's Perfect Pass Competition: tosser from across the net, passer passes to a target standing on a box at the net. Target catches = 1 point. Trains accuracy with immediate binary feedback.

## Key claims / ideas
- Toss from across the net, pass to target on a box at the net
- If target catches ball, passer scores a point (pass was "perfect")
- Binary feedback (catchable vs. not) drives accuracy focus
- High-rep, fun competitive format
- Works for beginner through HS/club

## Topics covered
- Perfect-pass scoring (catchable)
- Box-target passing drill
- Binary-feedback accuracy training
