# 4-2 volleyball system: A simple offense that works

## Summary
Tod Mattox (The Bishop's School) argues the 4-2 is making a comeback because a front-row-setter-in-all-six-rotations system is easy to learn and can be deadly when it features the Slide and a creative setter who attacks the second ball.

## Key claims / ideas
- 4-2 is easier for less experienced players to learn than 5-1 or 6-2.
- Front-row setter in all rotations frees the right back to focus on defense.
- A back-row Bic or similar set removes congestion from the middle.
- Slide is the featured weapon; defending a good Slide opens up other attacks.
- Setters should attack frequently, set the Slide, and run the Go (fast left-side).
- OH gets better looks because opposing middle must respect the Slide.
- Passers should target the middle of the court (not right side); right back stays deeper on transition to clear the Slide approach lane.
- Easy to test: run 6v6 scrimmage with both sides in 4-2, setters opposite each other.

## Topics covered
- 4-2 offensive system vs. 5-1 / 6-2 tradeoffs
- Slide attack as primary weapon
- Setter attack / dump
- Back-row attacking (Bic)
- Passing targets and defensive alignment to support the Slide
