# 3 circle sideout drill

## Summary
Ken Murczek's whiteboard-based sideout drill scores three rally outcomes per rotation: X = first-ball kill, / = won rally (no FBK), 0 = lost rally. Optional 0-3 pass rating underneath each circle surfaces the pass-to-point correlation.

## Key claims / ideas
- Three rallies per rotation, each logged on a whiteboard circle
- Scoring codes: X = FBK, / = rally win without FBK, 0 = rally loss
- Top outcome: all three circles = X (three FBKs)
- Layer in 0-3 pass rating below each circle to visualize pass-quality → outcome
- Great way to educate a team on the correlation between good passing and winning the point
- Rotate and restart after three rallies
