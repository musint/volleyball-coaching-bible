# Ideal hand path when blocking

## Summary
Tod Mattox (The Bishop's School) teaches blockers to slide their hands over the net instead of going up-then-forward, using a homemade cardboard-cylinder-and-elastic tool to groove the path for more touches and blocks.

## Key claims / ideas
- Ideal blocking hand path slides over the net rather than going straight up and then forward.
- Sliding keeps the hands over the net longer, producing more touches and more blocks.
- Mattox uses a homemade training tool (cardboard cylinder + elastic) to cue the correct path.
