# Match-Prep — In-Match Stat Collection Sheet

## Purpose

A single-page, paper-and-pen template for a non-coach helper — manager, assistant, or parent volunteer — to collect the **decision-grade stats** the head coach actually uses between rallies and at timeouts. Per Kiraly's framing, fewer-and-better beats more-and-better-tagged: the operational shortlist for in-match decisions is passing quality, side-out / FBSO, hitting efficiency, serving aggressiveness, and blocking touches. Everything else is post-match diagnostic and belongs on the [[match-prep-video-review-workflow]] sheet.

The collector does not need to know volleyball deeply. They need to know **what mark goes in what box** and to keep up with the rally pace. Calibrate them before warmups: walk through the legend, run a 10-rally dry pass, and confirm they can mark a passing rep in under three seconds. See [[match-prep]] for how this sheet feeds in-match adjustments.

## Inputs

- Roster with jersey numbers (top of sheet).
- Starting rotation (Setter in zone — circle one of 1/2/3/4/5/6).
- Opponent name + match date + set number.
- A clipboard, two pens (one runs out), and a stopwatch or phone.
- 10-minute calibration with the head coach pre-match: legend review + dry-pass reps.

## Form

**Per-set sheet — print one copy per set.**

```
OPPONENT: ________________   DATE: ___________   SET #: __   STARTING ROT (S in zone): 1 2 3 4 5 6
```

### A. Per-player tally — passing, hitting, serving, blocking

Mark a slash for each rep. Aces go in the Ace column; service errors in the Err column.

|   |           |        |        |        |        |       |       |       |         |        |         |          |          |         |
|   |           |        |        |        |        |       |       |       |         |        |         |          |          |         |

|   |           |        |        |        |        |       |       |       |         |        |         |          |          |         |

**Legend (for the collector):**
- **Pass 3** — perfect pass, setter has all options
- **Pass 2** — setter to target, options reduced (no quick)
- **Pass 1** — off-system, setter forced off the net or off-position
- **Pass 0** — overpass, ace, or unplayable
- **Hit K / E / 0** — kill / hitting error (out, net, blocked-for-point) / in-play (no kill, no error)
- **Srv Ace / In / Err** — ace / in-play serve / service error (net, out, foot-fault)
- **Blk Asst / Kill / Err** — block touch that allows a teammate dig / direct stuff block for a point / blocking error (net, reach-over, ball over the antenna)

### B. Per-rotation side-out tracker

For each rotation our team is in (Setter zone listed), mark **W** when we side-out on the first attack of the rally (FBSO win), **L** when we don't. Tally at the end of the set.

| Setter in | Rallies (W / L per rally, in order)            | FBSO W | FBSO L | FBSO % |
|-----------|------------------------------------------------|--------|--------|--------|
| Zone 1    |                                                |        |        |        |
| Zone 2    |                                                |        |        |        |
| Zone 3    |                                                |        |        |        |
| Zone 4    |                                                |        |        |        |
| Zone 5    |                                                |        |        |        |
| Zone 6    |                                                |        |        |        |

**FBSO % = FBSO W / (FBSO W + FBSO L).** Team-level FBSO indicator at the bottom: highlight any rotation under 40% with a star — that is the rotation the head coach needs to know about at the next timeout.

### C. Set summary (filled at end of set, handed to coach)

```
Set _: WON / LOST   Score: _____ to _____
Team passing avg (3+2+1+0 / total reps × ...): ______
Team hitting eff ((K - E) / total attempts): ______
Team FBSO %: ______       Worst rotation (FBSO): Zone __  ( ___ %)
Server who scored most points: # ____   Server with most errors: # ____
Hot opposing hitter (most kills against us): # ____ (___ K)
```

## Workflow

1. **Pre-match (10 min).** Coach reviews the legend with the collector, confirms jersey-number map, and walks through three dry passing reps + one full rally tally.
2. **In-match.** Collector tallies per-rep on Sheet A, marks W/L per rally on Sheet B. The collector is **eyes-down, no commentary** — speaking up while marking misses the next rep. Bench assistant (not the collector) is the in-match-tactics watcher per [[match-prep]].
3. **Between sets (90 sec).** Collector hands Sheet A and Sheet B summaries to the head coach (or assistant). Star-flag rotations under 40% FBSO. Coach scans for: worst-passing player (sub or move out of receive zone), hot opposing hitter (block adjustment per [[aoc-2024-kiraly-in-game-tactics]]), and our serving rotation strength (hold servers back vs. push them).
4. **Post-match.** Compile per-set sheets into a match-level summary. Pass-quality and hitting-efficiency entries get logged into the season trend file; rotation FBSO numbers feed the next opponent's [[match-prep-opponent-tendency-form]] entry on us.
5. **Edge cases.** If the collector falls behind, **drop the per-rotation tracker first** and protect the per-player passing column — passing is the single biggest lever on the offense and the highest-priority stat.
